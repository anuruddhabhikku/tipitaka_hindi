# tipitaka_hindi

## 📚 User Guide

The Tipitaka website includes a comprehensive bilingual user guide. You can view it online at:

**[👉 Read the User Guide](https://anuruddhabhikku.github.io/tipitaka_hindi/user_guide.html)**

### Quick Features:
- **Hindi Translation**: Click 📖 icon or use two-finger tap on mobile
- **Dictionary Lookup**: Click any Pali word for instant meaning
- **Commentary**: Click 📚 icon for deeper explanations  
- **Keyboard Shortcuts**: `Ctrl+H` to hide/show Pali, `Ctrl+←/→` to navigate
- **Dark Mode**: Toggle from floating menu (⋮)

For complete documentation, visit the [User Guide](https://anuruddhabhikku.github.io/tipitaka_hindi/user_guide.html) on the live site.
